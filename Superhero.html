About: This was an assignment where we were tasked to make a Java quiz. We did this by using Filewriter to get the chocies from a text file and output the results to another text file.

Challenges: Since this was one of the first few days working with Java,
I was not familiar with certain methods,such as how to actually use
a String file, buffered reader, or Scanner(System.in). So it was just
trying to figure out how to output the results to another file, while also
getting used to the structure of java, that made this assignment a
milestone.

Accomplishments: I am proud that I got to create a game,
that anybody could use. In it you would first enter your name,
and you will see the options and each question afterwards.
When you finish, there is a motivational statement outputted,
where you could even see your name afterwards.
